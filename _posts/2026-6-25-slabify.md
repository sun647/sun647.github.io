## How to run slabify?   
1. Install as instructed (https://github.com/CellArchLab/slabify-et)
2. Create folders for masks and masked tomograms

```bash
    mkdir -p slabify/masks slabify/masked_tomo
```

3. Test on one tomogram. Adjust the offset to cover the entire lamella

```bash
    slabify -i L9_ts_002_10.00Apx.mrc -o slabify/masks/L9_ts_002_10.00Apx_mask.mrc --output-masked slabify/masked_tomo/L9_ts_002_10.00Apx_maskedtomo.mrc --angpix 10 --offset 30 --measure
```

4. Run for all tomograms in the reconstruction folder

```bash
    for tomo in *.mrc; do
    base="${tomo%.mrc}"
    slabify \
        -i "$tomo" \
        -o "slabify/masks/${base}_mask.mrc" \
        --output-masked "slabify/masked_tomo/${base}_masked.mrc" \
        --angpix 10 \
        --offset 50 \
        --measure
    done
```
